SQL命令

发表于 2015-08-01   |   分类于 数据库   |     |   阅读次数

用于与关系型数据库交互的标准 SQL 命令有 CREATE、SELECT、INSERT、UPDATE、DELETE 和 DROP，这些命令按用途分成如下几组：

数据定义语言(DDL)

命令 描述
CREATE 创建新的表、视图或者其他数据库中的对象
ALTER 修改现存数据库对象，比如一张表
DROP 删除表、视图或者数据库中的其他对象

数据操纵语言(DML)

命令 描述
SELECT 从一张或者多张表中检索特定的数据
INSERT 创建一条新记录
UPDATE 修改记录
DELETE 删除记录

数据控制语言(DCL)

命令 描述
GRANT 赋予用户特权
REVOKE 收回赋予用户的特权
